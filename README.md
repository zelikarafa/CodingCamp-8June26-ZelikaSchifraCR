# Budget Buddy 🌷

Budget Buddy is a simple and cute expense tracking web application that helps users record their daily spending, view total balance, sort transactions, and visualize spending by category using a pie chart.

This project is built using HTML, CSS, JavaScript, and Chart.js.

## Features

* Add new spending transactions
* Input item name, amount, and category
* View transaction list
* Sort transactions by amount
* Delete transactions
* Display current balance
* Show spending by category using a pie chart
* Toggle light and dark theme
* View monthly summary
* Responsive and clean pastel UI

## Tech Stack

* HTML
* CSS
* JavaScript
* Chart.js
